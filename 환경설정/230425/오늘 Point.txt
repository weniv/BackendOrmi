* github 명령어
git clone repo주소 .		: 현재 폴더에 repo를 복사해 놓습니다. 최초에 한 번 실행합니다.
git pull 			: github에서 코드를 내려받는다.
git add . 			: 내 파일을 stage에 올려놓는다.
git commit -m '메시지'	: '메시지'를 메시지로 새로운 버전을 만듭니다.
git push			: 만든 버전을 GitHub에 올립니다.
